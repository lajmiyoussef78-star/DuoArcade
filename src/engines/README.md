# Engines

Each game is a plain JS module with the same tiny interface as before —
your original engine files from `engines/` drop in here **unchanged**.

Already implemented in this build (verified against test.mjs):
  ttt.js · connect4.js · dots.js

Placeholders waiting for your originals:
  reversi.js · gomoku.js · memory.js · pong.js · sketch.js ·
  wordrace.js · maze.js · reflex.js

After copying your files in, run `npm run test:engines` to verify.
